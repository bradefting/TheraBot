theraBots Features

Some of the things theraBot can do are:

  • Pre-Qualify Patients for NRN program
  • Display slideshow of T-Rec program
  • Allow questions to be answered using easy-to-click buttons
  • Allow users to exit at anytime by entering EXIT
  • Allow users to end conversation anytime by entering QUIT

About the Bot

The idea to create a bot that is used to pre-qualify patients for therapy programs was Selena Efting's. Selena thought her brain and spinal cord injury hospital would benefit from having theraBot pre-qualify patients to decrease the amount of time required to review patient applications for programs like the NeuroRecovery Network (NRN) program.
